## Python Cheat Sheet

This cheat sheet covers some of the basics of Python programming, including data types, operators, control flow, functions, and more.

### Data Types

* **Numbers:** Integers (e.g., 42), floating-point numbers (e.g., 3.14), complex numbers (e.g., 3+2j)
* **Strings:** Text data enclosed in quotes (e.g., "Hello, world!")
* **Booleans:** True or False values
* **Lists:** Ordered collections of items enclosed in square brackets `[]` (e.g., `[1, 2, 3, "apple"]`)
* **Tuples:** Similar to lists but immutable (cannot be changed after creation)
* **Dictionaries:** Unordered collections of key-value pairs enclosed in curly braces `{}` (e.g., `{"name": "Alice", "age": 30}`)

### Operators

* **Arithmetic:** `+`, `-`, `*`, `/`, `//` (integer division), `%` (modulo), `**` (exponentiation)
* **Comparison:** `==`, `!=`, `<`, `>`, `<=`, `>=`
* **Logical:** `and`, `or`, `not`
* **Assignment:** `=`, `+=`, `-=`, `*=`, etc.

### Control Flow

* **if statements:** Execute code based on a condition
* **elif statements:** Optional alternative conditions
* **else statements:** Code to execute if none of the above conditions are met
* **for loops:** Iterate over a sequence of items
* **while loops:** Repeat a block of code as long as a condition is true

### Functions

* Define reusable blocks of code with `def`
* Can take arguments and return values

### Strings

* Indexing and slicing to access characters or substrings
* String methods like `upper()`, `lower()`, `split()`, `join()`

### Lists and Tuples

* Access elements by index `[]`
* Slicing to extract portions of the list/tuple
* List methods like `append()`, `insert()`, `remove()`, `sort()`

### Dictionaries

* Access elements by key `{}`
* Add or modify key-value pairs
* Dictionary methods like `get()`, `keys()`, `values()`, `items()`

### Getting More Help

* Refer to online resources like [https://docs.python.org/](https://docs.python.org/) for detailed information
* Explore interactive tutorials and online courses
