## JavaScript Cheat Sheet

**Data Types:**

* **Numbers:** Represent numeric values (e.g., 10, 3.14).
* **Strings:** Represent textual data enclosed in quotes (e.g., "Hello").
* **Booleans:** Represent true or false values (true/false).
* **Arrays:** Ordered collections of items of any data type (e.g., `[1, "apple", true]`).
* **Objects:** Unordered collections of key-value pairs (e.g., `{ name: "John", age: 30 }`).

**Variables & Operators:**

* **Declaring Variables:** Use `let`, `const`, or `var` (avoid `var` for modern JavaScript) followed by the variable name (e.g., `let name = "Alice"`).
* **Operators:** Perform calculations (`+`, `-`, `*`, `/`) or comparisons (`==`, `!=`, `<`, `>`) on values.

**Control Flow:**

* **Conditional Statements:** Use `if`, `else if`, and `else` to control code execution based on conditions.
* **Loops:** Use `for`, `while`, and `do...while` loops to repeat code blocks a certain number of times or until a condition is met.

**Functions:**

* **Defining Functions:** Create reusable blocks of code with `function functionName(parameters) { /* code here */ return value; }`.
* **Calling Functions:** Execute a function using its name followed by parentheses (e.g., `functionName()`).

**DOM Manipulation:**

* **Accessing Elements:** Use `document.getElementById("id")` or `document.querySelector("selector")` to get references to HTML elements.
* **Modifying Elements:** Change element content with `.textContent = "new text"` or styles with `.style.property = "value"`.

**Common JavaScript Functionality:**

* **Events:** Listen for user interactions (clicks, scrolls, etc.) using `element.addEventListener("event", function)`.
* **Alerts & Prompts:** Use `alert("message")` to display a message box or `prompt("message", "default text")` to get user input.
* **Console Logging:** Use `console.log("message")` to print messages to the browser's developer console for debugging.
